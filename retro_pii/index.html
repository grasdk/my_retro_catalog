Pentium III 500MHz
Uber scene and game machine with GUS, AWE32 with OPL3 chip, Voodoo2 and TNT2. All in a modern case with a window in the side to behold the hardware in action. An alternative Pentium II 333MHz can replace the PIII in the same setup. The machine is meant for demo scene and late DOS and early Windows game compatibility.
