486 DX2 66MHz
Bought in Engesvang in 1998 for nostalgia. Original harddisk still inside, but disconnected, in favour of a 4GB early SSD. The BIOS could not read it, so using XTIDE on the network card, to make the computer recognize the disk. Repairs on the motherboard (see photos) was done before I bought it. Bought to have a DOS compatible machine back when Windows 9x was on the rise.
