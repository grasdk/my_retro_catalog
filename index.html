Bought in Engesvang in 1998 for nostalgia. Original harddisk still inside, but disconnected, in favour of a 4GB early SSD. The BIOS could not read it, so using XTIDE on the network card, to make the computer recognize the disk. Repairs on the motherboard (see photos) was done before I bought it. Bought to have a DOS compatible machine back when Windows 9x was on the rise.

SFF mikrotower XP machine with 64Bit OS specs, but with Windows XP 32 bit installed, to run the most demanding XP games. The SFF case has only a single slot space for a GPU, so chose the Quadro K2000, which has official XP drivers and is one of the latest and fastest GPU's for XP. Only surpassed by the K2200, which does not have official XP drivers, and the GTX 750 which is very rare as a single slot card.
